Summary of changes:
1.  **ctrl+w support**: Added logic in `TypingScreen.on_key` to clear the input field when `ctrl+w` is pressed.
2.  **Refactored WPM/Accuracy**: Implemented Net WPM (Gross WPM - Errors per minute) and proper accuracy calculation based on total keystrokes and errors.
3.  **Expanded word lists**: Added `ALICE` (Alice in Wonderland) and `PRIDE` (Pride and Prejudice) word lists to `words.py`.
4.  **Interactive Menu**: Created `MenuScreen` with `OptionList` for selecting test parameters.
5.  **Updated TypingApp**: Integrated `MenuScreen` and added `start_custom_test` method.
6.  **CLI Updates**: Modified `__main__.py` to show the menu by default and added a `serve` subcommand.
7.  **Cursor-aware display**: Refactored `TypingScreen._render_display` to show 3 lines of words with the current word centered.
8.  **SSH Server**: Added `asyncssh` dependency and a basic `server.py` module.
9.  **Testing**: Added `tests/test_logic.py` and fixed some bugs in `words.py`.
